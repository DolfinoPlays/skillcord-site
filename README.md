# Skillcord Website

Docusaurus site for the Skillcord bot.

## Quick start

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```

## Replace before launch

- `YOUR_INVITE_LINK_HERE`
- `YOUR_SUPPORT_SERVER_LINK_HERE`
- `YOUR_GITHUB_REPO_URL_HERE`
- `YOUR_BOT_NAME_OR_OWNER`
- `YOUR_GITHUB_USERNAME`

## GitHub Pages notes

- For a project site, leave `url: 'https://YOUR_GITHUB_USERNAME.github.io'` and `baseUrl: '/skillcord-site/'`
- For a custom domain like `https://skillcord.gg`, change to:
  - `url: 'https://skillcord.gg'`
  - `baseUrl: '/'`

After switching to a custom domain, rebuild and redeploy.
