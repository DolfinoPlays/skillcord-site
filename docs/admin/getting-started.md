# Getting started

This is the basic setup flow for a new Skillcord server.

## 1. Invite the bot

Use the public invite link:

[Invite me!](https://discord.com/oauth2/authorize?client_id=1337481384674984026)

Make sure the bot has the permissions it needs to manage roles and edit channel or category permission overwrites where applicable.

## 2. Create your first branch

Start simple.

A good first tree usually has:

- 2 starting branches
- 1 to 3 follow-up unlocks per branch
- low early skill point costs
- clear node names and descriptions

## 3. Configure your rewards

For each node, decide what it does:

- grants a role
- unlocks a category or channel
- only unlocks future nodes

## 4. Test with an alt account

Before announcing your tree, test it with a second account in your server.

Check:

- XP gain
- level-up rewards
- point spending
- role assignment
- category or channel visibility
- prerequisite locking

## 5. Publish your tree

Once your test is clean, make the tree visible to members and let them start progressing.

## Suggested starter layout

- Branch A: Community
- Branch B: Content
- Community unlocks social roles and hidden hangouts
- Content unlocks creator tools, submission channels, and showcase areas
