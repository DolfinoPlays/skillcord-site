# Commands

These are placeholder command names for your public docs. Adjust them to match your final bot.

## Member commands

- `/profile` — view XP, level, skill points, and unlocked nodes
- `/tree` — open the server's skill tree
- `/unlock <node>` — unlock a node with skill points
- `/points` — show available skill points
- `/respec` — reset your tree if enabled

## Admin commands

- `/skillcord setup` — begin guided setup
- `/skillcord branch create` — create a branch
- `/skillcord node create` — create a node
- `/skillcord node edit` — edit a node
- `/skillcord node delete` — delete a node
- `/skillcord preview` — preview the live tree
- `/skillcord validate` — check config and permission issues
- `/skillcord export` — export server config
- `/skillcord import` — import server config
