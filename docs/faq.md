# FAQ

## Is Skillcord only for role unlocks?

No. A node can grant roles, unlock channels or categories for a specific member, or simply unlock future nodes.

## Do I need a role for every node?

No. You can mix role nodes with access nodes and progression nodes.

## Can I make multiple starting branches?

Yes. Skillcord is built for branching trees.

## Does each server get its own config?

Yes. Every server has its own tree and progression data.

## Can I use Skillcord without hidden channels?

Yes. You can use it only for roles and progression if you want.

## Can members unlock both branches?

That depends on how the server owner designs the tree. A tree can allow multiple paths or force exclusive choices.
