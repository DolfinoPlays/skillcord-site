# How it works

Skillcord is built around a few core steps:

1. Members earn XP by participating in your server.
2. XP becomes levels.
3. Level-ups can grant skill points.
4. Members spend skill points on nodes in your tree.
5. Nodes apply their configured rewards.

## Per-server configuration

Every server has its own config. That means:

- its own XP settings
- its own branches
- its own prerequisites
- its own role and channel targets
- its own balance and progression style

## Personal progression

A member's unlocked nodes are tracked per server. Someone can be deep into one tree in one server and have zero progress in another.

## Flexible tree design

You can make:

- simple two-branch trees
- class systems
- faction unlock systems
- content progression ladders
- optional role and content hubs
