# Skillcord

Skillcord turns your Discord server into a progression system.

Members gain XP, level up, earn skill points, and spend those points on a branching unlock tree. A node can unlock a role, unlock a channel or category, or simply unlock the next step in the tree.

## Main idea

Most leveling bots stop at XP and level rewards. Skillcord adds:

- branching paths
- unlock prerequisites
- role rewards
- optional access unlocks
- server-specific progression trees

## Node types

A node can be used for:

- **Role unlocks**: grants one or more Discord roles
- **Access unlocks**: unlocks a channel or category for a specific user
- **Progression unlocks**: unlocks future nodes without directly changing Discord permissions

## Use cases

- class systems
- faction systems
- gated community channels
- roleplay progression
- creator/supporter unlock paths
- knowledge or contribution trees
